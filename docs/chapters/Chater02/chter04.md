# 第02章：梯度下降 - 第4部分
## 2.4 双变量参数场景（参考课堂内容理解）  
### 核心内容：简化场景下的梯度下降逻辑  
| 场景拆解       | 具体内容                                                                 |
|----------------|--------------------------------------------------------------------------|
| 参数假设       | 模型仅含两个参数：θ = {θ₁, θ₂}（如两个特征的权重）                       | 
| 损失函数形式   | 损失值由θ₁和θ₂共同决定，记为L(θ)（如L(θ₁,θ₂)=θ₁²+2θ₂²）                   |
| 优化目标       | 找到(θ₁*, θ₂*)，使L(θ₁*, θ₂*)最小（3D曲面上的最低处）                     | 
| 计算逻辑       | 1. 初始(θ₁,θ₂)→算L(θ)；2. 求梯度(∂L/∂θ₁, ∂L/∂θ₂)；3. 沿负梯度更新         |
