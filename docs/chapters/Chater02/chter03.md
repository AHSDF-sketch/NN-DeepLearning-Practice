# 第02章：梯度下降 - 第3部分
## 2.3 梯度下降核心定义  
### 核心内容：梯度下降的严格定义与目标  
| 定义维度       | 具体表述                                                                 | 
|----------------|--------------------------------------------------------------------------|
| 官方定义       | 沿“损失函数的负梯度方向”迭代更新模型参数，求解损失函数极小值的优化方法     | 
| 核心目标       | 调整参数（w、b等），使“衡量预测误差的损失函数”达到最小                   | 
| 关键约束       | 每次迭代仅沿“负梯度方向”移动，确保损失值减小最快                         | 
| 数学本质       | 用局部梯度信息逼近极小值，属于“贪心算法”                                 |
