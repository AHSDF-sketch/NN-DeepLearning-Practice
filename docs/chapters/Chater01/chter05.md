# 第01章：梯度下降 - 第5部分
## 1.5 典型应用：无人机最短路径优化（对应PPT“无人机最短路径优化”模块）  
### 核心内容：梯度下降的工程应用逻辑  
| 应用拆解       | 具体内容                                                                 | 
|----------------|--------------------------------------------------------------------------|
| 应用场景       | 无人机从起点到终点，避开障碍物且路径最短（物流、巡检）                   | 
| 目标函数构建   | 总损失 = 路径长度 + 障碍物惩罚（距离越近惩罚越大）                       | 
| 优化过程       | 1. 初始化路径点；2. 算梯度；3. 沿负梯度调路径；4. 重复至收敛             | 
| 应用优势       | 比枚举法快10~100倍，支持实时调整                                         |
